This project focuses on analyzing customer purchasing behavior and sales trends using Excel, SQL, and Power BI.
The dashboard provides insights into revenue, customer activity, product performance, and regional trends.

# Objectives

Analyze overall sales performance
Identify top customers and products
Track monthly and yearly sales trends
Understand customer purchasing patterns

#Tools & Technologies Used

Excel → Data preparation and cleaning
MySQL (SQL) → Data querying and analysis
Power BI → Data visualization and dashboard creation
Dataset Information

#The dataset includes the following fields:

Order_ID
Order_Date
Customer_Name
Region
Product
Category
Quantity
Price

# Key Metrics (KPIs)
Total Sales
Total Orders
Total Quantity Sold
Profit (calculated as 20% of sales)

# Dashboard Features
Sales by Region (Bar Chart)
Monthly Sales Trend (Line Chart)
Category-wise Sales (Pie Chart)
Top Products Analysis
Interactive Filters (Region, Category, Date)
SQL Analysis

Total Revenue Calculation
